Monitors the keyword density of keywords you select in the whole blog and gives a live update of the keyword density of the post as you type.

== Description ==

The plug-in allows you to choose as many keywords you wish to monitor for their density in your blog. This is to aid you in your SEO, to ensure the density of your keywords is optimized.

The plug-in monitors the keywords across the whole site, giving you an indication on the dashboard and a live indication while you type your post.

The plugin is intended to give you a guide on your keyword density and to help you with your SEO. In it's self it will not improve you rankings, but as a tool used correctly keyword density can improve your chances of a beter ranking.

Future:

We will improve this plugin by giving you stats on your keywords in page titles, alt tags, image names, page names, heading tags, wordpress tags and more.


== Installation ==

Upload the file via the wordpress install interface or Extract the zip file and just drop the contents in the wp-content/plugins/ directory of your WordPress installation and then activate the Plugin from Plugins page.
